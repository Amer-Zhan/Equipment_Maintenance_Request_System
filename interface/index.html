Database-Oriented Prototype for Equipment and Workover Request Tracking

This interface demonstrates a prototype information system designed for LLP “Service Oil Company Leopard”. The purpose of the system is to organize information about technical equipment, employees, maintenance requests, workover-related service tasks, statuses, and maintenance history.

The full project includes a PostgreSQL database, SQL scripts, Python scripts, static HTML/CSS pages, a GitHub-ready structure, and technical documentation for an Industrial Practice report.
