# Screenshots Guide for Industrial Practice Report

Use this folder to store screenshots that demonstrate the technical work completed during the project.

Recommended screenshots:

1. **GitHub Repository Structure**
   - Show folders: sql, python, interface, screenshots, docs.

2. **README.md Preview**
   - Show project title, technologies, and launch instructions.

3. **PostgreSQL Database in pgAdmin**
   - Show `leopard_practice_db` in the pgAdmin browser panel.

4. **Tables in pgAdmin**
   - Show the created tables: departments, employees, equipment, request_statuses, workover_requests, maintenance_logs.

5. **ERD Diagram**
   - Show the database relationships between the six tables.

6. **SQL Query Result**
   - Run a query from `04_queries.sql`, for example active requests or dashboard summary.

7. **Python Code in VS Code**
   - Show one of the Python files such as `main.py`, `equipment_operations.py`, or `reports.py`.

8. **Python Script Output in Terminal**
   - Run `python main.py` and show output from one menu option.

9. **Dashboard HTML Page**
   - Open `interface/dashboard.html` in a browser.

10. **Equipment HTML Page**
    - Open `interface/equipment.html` in a browser.

11. **Requests HTML Page**
    - Open `interface/requests.html` in a browser.

12. **Project Folder in VS Code**
    - Show the whole project opened in VS Code.

Tip: name screenshots clearly, for example:

- `01_github_repository.png`
- `02_pgadmin_tables.png`
- `03_erd_diagram.png`
- `04_sql_query_result.png`
- `05_python_terminal_output.png`
- `06_dashboard_page.png`
